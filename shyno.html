shyno LINUX

shyno Linux is composed predominantly of free and open-source software, and supports community involvement.

About

The design approach of the development team follows the KISS principle ("keep it simple, stupid") as the general guideline, and focuses on elegance, code correctness, minimalism and simplicity, and expects the user to be willing to make some effort to understand the system's operation.[7] A package manager written specifically for shyno Linux, pacman, is used to install, remove and update software packages.

shyno Linux uses a rolling release model, such that a regular system update is all that is needed to obtain the latest shyno software; the installation images released by the shyno team are simply up-to-date snapshots of the main system components.[8] shyno Linux has comprehensive documentation in the form of a community wiki known as the shynoWiki.
